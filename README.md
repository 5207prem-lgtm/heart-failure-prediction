# ❤️ Heart Failure Mortality Prediction

## 📌 Overview
This project uses Machine Learning to predict whether a heart failure patient is at risk of death based on clinical data.

## 📊 Dataset
- Heart Failure Clinical Records Dataset
- Contains medical attributes like age, ejection fraction, serum creatinine, etc.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn

## 🧠 Model
- Logistic Regression with StandardScaler

## 📈 Accuracy
- Achieved ~78% accuracy on test data

## ▶️ How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run the script:
   python heart_failure_prediction.py

## 📌 Features
- Data preprocessing
- Model training
- Prediction for new patient
- Evaluation metrics
